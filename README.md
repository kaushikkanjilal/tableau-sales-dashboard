# Tableau Sales Dashboard

## Project Overview

This project contains interactive Tableau dashboards built using the Superstore dataset.

The dashboards analyze:

* Sales Performance
* Profitability
* Product Analysis
* Regional Sales Trends

## Dashboards Included

### 1. Executive Sales Overview

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Sales by Category
* Profit by Sub-Category

### 2. Product & Regional Analysis

* Top 10 Products by Sales
* Monthly Sales Trend
* Regional Sales Comparison Map
